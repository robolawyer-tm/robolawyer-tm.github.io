# robolawyer-tm

> Modeling beneficial social outcomes through schema-free semantic data, local-first computation, and human-centric security — all running on the hardware you may already own.

## About

robolawyer-tm is an open research ecosystem for predicting beneficial outcomes in complex human situations using analogical synthesis — bridging semantic (emotional/logos) and digital (analytical/code) processing on local, privacy-preserving hardware.

## Core Concepts

- **Analogical Synthesis**: Synthesis from thesis without antithesis — analogical, empathy-native output supported by analytical power, not dialectically opposed.
- **Vivify Engine**: Schema-free semantic data engine using autovivification (Perl-inspired hash-of-hashes). Categories emerge from keyword co-occurrence, not predefined schemas.
- **FABRIC**: The component model — Vivify, Payload, Secrecy, Freeze, Server. The pipeline for creating and retrieving vivified data.
- **Left/Right Duality**: Left (semantic/logos/emotional) and Right (digital/analytical/code) processing pipeline for dual-lens inference.
- **Social Neurology Framework**: Layer 1-4 model (neurological substrate → social/relational → ideological → institutional) explaining how control behaviors propagate through institutions without conscious coordination.
- **Polarization Signal**: Bell curve → rabbit ears distribution shift — tail populations drive policy as institutional middle loses voice. Measurable in text.
- **Non-Verbal Distal Communication**: Emotional and neurological signal carried in formal text across time and distance. Recoverable from legislative and court language.
- **Functional Art as Output**: Target output operates at Layer 1 — holds irresolvable tension without premature resolution. Person changed, not problem solved.
- **Inference Pipeline**: Text → keyword clump extraction → co-occurrence graph → emergent categories → autovivified filesystem + JSON → tension scoring → beneficial outcome prediction.
- **YTDMSP**: Yet-To-Define Memory/Storage Paradigm — captures distilled essence of LLM mid-inference reasoning across sessions. Transforms one-shot analysis into lifelong pattern recognition.
- **Local-First**: Zero cloud dependencies. All data on participant hardware. SSH tunnels. Filesystem-as-database.
- **Star-Bridge Topology**: Android phones as hardened edge servers (Termux/Flask), connected via SSH mesh networking.
- **Justice Systems Application**: Family court as primary testbed — clean timestamped data, documented divergence from NIH developmental science, measurable Layer 1-4 signal in legislative text.
