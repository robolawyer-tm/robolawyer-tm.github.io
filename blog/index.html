Dimensional Fields — Topics

Ideas organized by topic, not date. Living documents — each grows as the project develops.

  • The Status Quo Operator

    When truth is not enough. Structural commitment preservation — why institutions cannot release prior commitments even when they know the truth. DNA exoneration as the cleanest signal. Navigational modeling target.

  • The Logos Schema

    A coordinate system for social communication. Functional schema (what communication does) and structural schema (where it occurs in social space) held in tension. The structure/function dichotomy as a second orthogonal axis alongside analog/digital.

  • Operators: Theory to Code to LLM

    How researcher frameworks become computational operators — and why researcher names in the schema are query seeds for living theory, not just citations. The analog-to-digital bridge at the semantic level.

  • Social Neurology

    The layer below speech. Layer 1-4 framework, rabbit ears polarization, empathy as the generative force that created social structure rather than the glue that holds it together.

  • Evolution as Pillar

    The inference store as selection pressure. Hypotheses enter sparse; the data selects. RISC discipline reframed: not "only add what's proven" but "add sparse hypotheses and let the data decide." The schema is the genome.

  • Field Data & the Inference Store

    Naming the input and what the pipeline builds from it. Why "field data" does user-facing work that "corpus" cannot. The inference store as both output and selection environment.

  • Cloudflare Architecture

    The production target — Workers, KV, Durable Objects, LLM consultation at the edge. Privacy by architecture: raw conflict data never reaches persistent storage. Parked until the Python prototype proves the model.
